# LarsiQR Studio

**LarsiQR Studio** is a modern desktop QR code generator for Windows.

It allows you to quickly generate QR codes for URLs or text, export them in multiple formats, and even create hundreds of QR codes automatically using a CSV batch system.

Designed with a clean **dark UI** and built using **Java Swing + FlatLaf**.

Created by **Knirpslarsi**

---

# Features

• Generate QR codes from URLs or text  
• Clean **dark interface**  
• **Live preview with autoscaling**  
• Export formats:
- PNG
- JPG
- SVG  

• **Logo support** for single QR codes  
• **Batch QR code generation via CSV**  
• **Drag & Drop support**
- Drop text or URLs
- Drop CSV files to start batch generation  

• Built-in **update checker**  
• Windows installer

---

# Installation

Download the latest installer from the **Releases** section.

Run the installer and follow the setup wizard.

After installation you can launch the program from:

• Desktop shortcut  
• Start menu  

---

# Generating a Single QR Code

1. Enter a **URL or text**
2. Click **Generate QR Code**
3. (Optional) click **Add Logo**
4. Choose export format
5. Click **Save QR Code**

---

# Batch QR Code Generator

LarsiQR Studio can automatically generate large numbers of QR codes using a CSV file.

Example use cases:

• product labels  
• event tickets  
• links for marketing  
• bulk QR generation

---

# Example CSV

Inside the application you can download a ready-to-use CSV file.

If the logo field is empty the QR code will be generated **without a logo**.

# Drag & Drop

Supported drag & drop actions:

| Type | Result |
|----|----|
URL | fills the input field |
Text | fills the input field |
CSV file | starts batch generator |
